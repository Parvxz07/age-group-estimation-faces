# Age Group Estimation from Faces
## Mohammed Parvez | 75924561 | Class 120B

## Project Description
Deep learning system to classify faces into 5 age groups using MobileNetV2 transfer learning.

## Age Groups
- Child (0-12)
- Teenager (13-17)
- Young Adult (18-35)
- Middle-Aged (36-60)
- Senior (60+)

## Dataset
UTKFace Dataset - 20,000+ face images

## Model
MobileNetV2 with custom classification head

## Results
- Test Accuracy: 76.4%
- Best class: Young Adult (F1: 0.83)

## Tools
Python, TensorFlow, Keras, OpenCV, Google Colab
